# Last Hope Memorial

A memorial website preserving the history of the **Last Hope** mod community - an X-COM inspired gaming project that spanned from 2002 to 2008.

## About Last Hope

Last Hope was a modding community with two distinct eras:

### Half-Life Era (2002-2006)
An X-COM inspired modification for Valve's Half-Life, featuring:
- Custom weapons modeled after the X-COM arsenal
- Tactical maps inspired by X-COM mission environments
- Active phpBB community forums

### Crysis Era (2007-2008)
A total conversion for Crytek's Crysis, featuring:
- Original sci-fi storyline about the **Earth Scope** (planetary defense grid)
- The **Slaver Horde** alien invasion
- The **United Earth Coalition (UEC)** resistance
- Class-based objective multiplayer gameplay
