Crysis Era

2007 - 2008 | Last Hope // Crysis Modification

Historical Note

This Crysis mod is not directly related to the original X-COM Last Hope project. After the original lasthopemod.com domain was abandoned, a separate team coincidentally chose the same name for their Crysis total conversion. While unrelated to the X-COM inspired Half-Life mod, we document this curious coincidence as part of the "Last Hope" naming legacy.

The true continuation of the original X-COM Last Hope project is X-COM: Last Hope Neo for Half-Life 2/Source.

The Crysis "Last Hope"

In 2007, a separate team created a total conversion for Crytek's Crysis under the name "Last Hope." With a completely new team and a fresh creative vision, this project set out to create an original sci-fi storyline about humanity's last stand against an alien invasion.

The Crysis mod featured the Earth Scope - a planet-wide orbital shield defense grid - the Slaver Horde alien invaders, and the United Earth Coalition (UEC) military forces fighting to reclaim humanity's home.
